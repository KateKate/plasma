# Button
Компонент для отображения кнопки. Может быть отображена в нескольких размерах и цветах, с текстом и иконкой или без оных.

## Props
<Props of={Button} />

## Примеры использования

### Базовое применение
<Playground>
  <ThemeBackground>
    <Button>Привет, мир!</Button>
  </ThemeBackground>
</Playground>

### Размер кнопки
Чтобы изменить размер кнопки, установите свойство ``size`` в одно из следующих значений: ``"l"``, ``"m"``, ``"s"``.
<Playground>
  <ThemeBackground>
    <Button size="l">Button</Button>
    <Button size="m">Button</Button>
    <Button size="s">Button</Button>
  </ThemeBackground>
</Playground>

### Вид кнопки
Чтобы изменить вид кнопки, установите свойство ``view`` в одно из следующих значений: ``"primary"``, ``"secondary"``, ``"warning"``, ``"critical"``, ``"checked"``, ``"clear"``.
<Playground>
  <ThemeBackground>
    <Button view="primary" size="s">Button</Button>
    <Button view="secondary" size="s">Button</Button>
    <Button view="warning" size="s">Button</Button>
    <Button view="critical" size="s">Button</Button>
    <Button view="checked" size="s">Button</Button>
    <Button view="clear" size="s">Button</Button>
  </ThemeBackground>
</Playground>

### Границы кнопки
Чтобы изменить границы кнопки, установите свойство ``pin`` в одно из допустимых значений.
<Playground>
  <ThemeBackground>
    <div style={{ display: 'flex', flexDirection: 'column' }}>
      <div style={{ display: 'flex', marginBottom: '10px' }}>
        <Button pin="square-square" size="s">Button</Button>
        <Button pin="circle-circle" size="s">Button</Button>
        <Button pin="clear-clear" size="s">Button</Button>
      </div>
      <div style={{ display: 'flex', marginBottom: '10px' }}>
        <Button pin="square-clear" size="s">Button</Button>
        <Button pin="clear-square" size="s">Button</Button>
      </div>
      <div style={{ display: 'flex' }}>
        <Button pin="circle-clear" size="s">Button</Button>
        <Button pin="clear-circle" size="s">Button</Button>
      </div>
    </div>
  </ThemeBackground>
</Playground>
